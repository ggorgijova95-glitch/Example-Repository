# Hyperlinks Homework Project

This project consists of three HTML documents: `home.html`, `about.html`, and `contact.html`, which are linked together using hyperlinks. Each document contains unique content and images that link to external websites.

## File Structure

```
hyperlinkshomework
├── home.html
├── about.html
├── contact.html
├── assets
│   └── images
└── README.md
```

## Description of HTML Files

### home.html
- Serves as the homepage.
- Contains a welcome message.
- Links to the About Us and Contact pages.
- Includes several images that link to external websites.
- Features a "back to the top" link at the bottom.

### about.html
- Provides information about the website or organization.
- Includes a brief description.
- Links back to the Home and Contact pages.
- Contains several images that link to external websites.

### contact.html
- Contains contact information (e.g., email address, phone number).
- Links to the Home and About Us pages.
- Includes several images that link to external websites.

## Assets
- The `assets/images` folder is designated for storing image files used in the HTML documents.

## Navigation
- Users can navigate between the pages using the hyperlinks provided in each document.
